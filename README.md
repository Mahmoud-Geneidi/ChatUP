ChatUP is a desktop messaging application built using Java, JDBC, MySQL, RMI, FXML, CSS and managed using Maven.

## Features

- Realtime Messaging
- Group Chats
- Media Sharing
- Chatbot
- Notifcation
- Admin Dashboard
- System Statistics
- Admin Announcements
